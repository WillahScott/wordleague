# WordLeague Game

__Friendly wordle leagues to play and challenge your friends on the popular Wordle-style word guessing.__


## Architecture

Backend: Supabase for db + subscriptions (realtime) + auth.
Frontend: Svelte + TailwindCSS.
Deployed on Vercel.
